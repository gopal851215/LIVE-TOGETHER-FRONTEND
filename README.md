# LIVE-TOGETHER Frontend (Vite + React + Tailwind)

## Quickstart
```bash
npm install
npm run dev
```

## Deployment to Vercel
1. Push to GitHub (`live-together-frontend`)
2. Import to Vercel dashboard
3. Add env: `VITE_API_URL=https://your-backend.onrender.com/api`
4. Deploy!

## Build Scripts
- `npm run build` → `dist/` folder
- `npm run preview` → local production preview

See root README.md for full stack deployment.
